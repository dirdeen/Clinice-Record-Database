## Group Members

## Project Description
This project implements a Public Health Clinic Records System using MySQL. The database manages patients, doctors, appointments, medical records, prescriptions, medications, departments, and payments.

## Technologies Used
- MySQL
- XAMPP
- phpMyAdmin

## Features
- Patient Management
- Doctor Management
- Appointment Scheduling
- Medical Records
- Prescription Tracking
- Payment Management
- User Management
